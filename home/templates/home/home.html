About Xarsed

Xarsed is a companion that helps you meet your needs in harmony with nature. In this path, it has moved towards one of today's needs, namely energy. In the first step, Xarsed has created an algorithm based on scientific data that helps you design a solar power plant based on your need pattern. This power plant is based on the most common electricity supply systems, photovoltaic panels and batteries.

Xarsed is open source and free. You can see the details of the algorithm in the Xarsed Python package GitHub repository and the overall structure of its web app in the Xarsed web application GitHub repository . You can raise any issue you have in its GitHub.

How to interact with Xarsed

Xarsed is designed in such a way that the user can easily interact with it. Xarsed always offers an alternative solution at each step so that, in the absence of the necessary data, you can continue the path. Each step in the Xarsed path has a practical guide that you can reviewing it. Xarsed suggests that, if this is your first time with it, use the alternative options to go through the path completely once. In this way, you will get a complete view of Xarsed's performance.

To get started, just press the button below to start the process. The only important point is that, when you start the path, a session is created for you in which your data is stored at each step. The expiration of this session comes 60 minutes after the start of the session, which is extended by each time you press new buttons on the path. Therefore, keep in mind that the interval between pressing the two buttons should be less than an hour, because after the session expires, your data will be deleted and you will have to start the journey over.
